Deploy a pelican project on Ubuntu/Debian development environment

As a note, I put how to setup pelican here, after 2 years using this static blog system, I still think it is one of the best option.

To setup the pelican project,

  1. Install openssl

    :::sh sudo apt-get install libssl-dev

  2. Update pip

    :::sh sudo pip install --upgrade pip

  3. Install fabric and bs4

    :::sh sudo pip install fabric bs4

  4. Install pelican and other dependencies

    :::sh sudo pip install pelican Markdown typogrify

  5. Check out or git clone the pelican themes and plugins

    :::sh git clone

for plugins: :::sh git clone https://github.com/getpelican/pelican-plugins.git

  1. install theme like

    :::sh sudo pelican-themes -i

  2. Checkout your pelican website code, and use fab build to check if the pelican code can run

    :::sh git clone cd fab build

Tips:

fab publish to send it remotely to the web server, the configuration is in the publishconf.py file.

To check the detail installation instruction, please refer to the official doc

Note:

Markdown must be installed to properly scan the md files.
